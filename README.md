# JNote

一款支持部分Markdown语法的轻量级便签应用，功能介绍如下

> * 支持笔记的添加、修改、删除等功能
> * 支持部分Markdown语法，可以快捷输入Markdown符号
> * 支持预览Markdown文本，并保存为本地图片
